# Cloud Computing HW-3

Static frontend for S3 website hosting.

## Repository Contents

- `index.html`
- `styles.css`
- `app.js`
- `.gitignore`

## S3 Upload Files

Upload only these files to your S3 bucket root:

- `index.html`
- `styles.css`
- `app.js`

## Run Locally

Open `index.html` in a browser.

## Notes

- This version is a static demo frontend.
- Upload data is browser-memory only in this phase.
